进度条 Progress #

给予用户当前系统执行中任务运行状态的反馈，多用于运行一段时间的场景，有效减轻用户在等待中产生的焦虑感。

基本用法 #

简单的进度条。

45%
<template>
  <div class="box">
    <sb-progress width="50%" :percent="45" />
  </div>
</template>

进度条状态 #

通过 status 指定进度条状态

10%
10%
10%
10%
10%
<template>
  <div class="box">
    <sb-progress width="50%" :percent="percent" status="normal" />
    <sb-progress width="50%" :percent="percent" status="success" />
    <sb-progress width="50%" :percent="percent" status="warning" />
    <sb-progress width="50%" :percent="percent" status="danger" />
    <div class="list">
      <sb-progress type="circle" :width="60" :percent="percent" status="normal" />
      <sb-progress type="circle" :width="60" :percent="percent" status="success" />
      <sb-progress type="circle" :width="60" :percent="percent" status="warning" />
      <sb-progress type="circle" :width="60" :percent="percent" status="danger" />
    </div>
    <sb-button @click="percent += 10">+10</sb-button>
    <sb-button style="margin-left: 10px" @click="percent -= 10">-10</sb-button>
  </div>
</template>

<script setup>
import { ref } from 'vue';
let percent = ref(10);
let status = ref('warning');
</script>

<style scoped>
.list {
  display: flex;
  justify-content: flex-start;
  align-items: center;
  margin: 20px 0;
}

.list > *:not(:first-child) {
  margin-left: 10px;
}
</style>

环形进度条 #

设置 type="circle" 将会展示环形进度条。

40%
<template>
  <div class="box">
    <sb-progress 
      :width="150" 
      :percent="40" 
      type="circle"
    />
  </div>
</template>

进度条大小 #

通过 size 设置进度条的大小。

10%
10%
10%

  <div class="box">
    <sb-progress width="50%" :percent="percent" size="small" />
    <sb-progress width="50%" :percent="percent" size="medium" />
    <sb-progress width="50%" :percent="percent" size="large" />
    <br />
    <sb-button @click="percent += 10">+10</sb-button>
    <sb-button style="margin-left: 10px" @click="percent -= 10">-10</sb-button>
  </div>
</template>

<script setup>
import { ref } from 'vue';
let percent = ref(10);
let status = ref('warning');
</script>

  offset: Percent,
  color: RGB | HEX

  <div class="box">
    <sb-progress 
      :width="500" 
      :percent="60" 
      :color="colorObj"
    />

      :width="100"

自定义进度条的颜色 #

可以 通过 color 设置进度条的颜色，通过 trackColor 设置剩余进度条的颜色。

60%

60%
<template>
  <div class="box">
    <sb-progress 
      width="50%" 
      :percent="percent" 
      :color="color" 
      :track-color="trackColor" 
    />
    <br />
    <sb-progress 
      type="circle"
      width="50" 
      :percent="percent" 
      :color="color" 
      :track-color="trackColor" 
    />
  </div>
</template>

<script setup>
const percent = 60;
const color = "#165dff";
const trackColor="#e7f3ff" 
</script>

API #

<progress> Props #

参数名描述类型默认值
type进度条的类型`'line''circle'`
size进度条的大小'small' | 'medium' | 'large''medium'
percent进度条当前的百分比number0
stroke-width进度条的线宽number-
width进度条的长度number | string-
color进度条的颜色string | array-
track-color进度条的轨道颜色string-
show-text是否显示文字booleantrue
status进度条状态'normal' | 'success' | 'warning' | 'danger'-
round进度条是否有圆角booleantrue
