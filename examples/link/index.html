链接 Link #

链接是你设置的引导用户进行跳转的组件

基本用法 #

链接的基本用法

link 
<template>
  <div class="box">
    <sb-link title="这是一个打开我们git的链接" href="https://github.com/aiai0603" target="_blank"> link </sb-link>
  </div>
</template>

类型配置 #

链接分为： default，secondary，success，warning，error 五种类型，你可以使用 type 属性配置他们。你也可以通过 class 自己配置你需要的色彩样式。

💡 说明

默认的状态主题包含正常状态、hover 状态和 disabled 状态的样式

default warning success error secondary 
<template>
  <div class="box">
    <sb-link>default</sb-link>
    <sb-link type="warning"> warning </sb-link>
    <sb-link type="success"> success </sb-link>
    <sb-link type="error"> error </sb-link>
    <sb-link type="secondary"> secondary </sb-link>
  </div>
</template>


<style scoped>
.box > *:not(:first-child) {
  margin-left: 10px;
}
</style>

下划线 #

设置 underline 属性可以让链接在鼠标悬停的时候显示下划线，下划线色彩和文字色彩统一。

default warning success error secondary 
<template>
  <div class="box">
    <sb-link underline>default</sb-link>
    <sb-link type="warning" underline> warning </sb-link>
    <sb-link type="success" underline> success </sb-link>
    <sb-link type="error" underline> error </sb-link>
    <sb-link type="secondary" underline> secondary </sb-link>
  </div>
</template>

<style scoped>
.box > *:not(:first-child) {
  margin-left: 10px;
}
</style>

禁用 #

设置 disabled 属性可以禁用链接。

default warning success error secondary 
<template>
  <div class="box">
    <sb-link disabled>default</sb-link>
    <sb-link type="warning" disabled> warning </sb-link>
    <sb-link type="success" disabled> success </sb-link>
    <sb-link type="error" disabled> error </sb-link>
    <sb-link type="secondary" disabled> secondary </sb-link>
  </div>
</template>

<style scoped>
.box > *:not(:first-child) {
  margin-left: 10px;
}
</style>
