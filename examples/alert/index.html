警告提示 Alert #

向用户显示警告的信息时，通过警告提示，展现需要关注的信息。

基本用法 #

展现需要关注的信息，适用于简短的警告提示。

This is an info alert.
<template>
    <div class="box">
        <sb-alert>This is an info alert.</sb-alert>
    </div>
</template>

提示类型 #

有 info，success，warning，error 四种类型的消息提示。normal 类型下默认不展示图标。

This is an info alert.
This is an success alert.
This is an warning alert.
This is an error alert.
<template>
  <div class="box">
    <div class="sb-row">
      <div class="sb-col">
        <sb-alert>This is an info alert.</sb-alert>
      </div>
      <div class="sb-col">
        <sb-alert type="success">This is an success alert.</sb-alert>
      </div>
      <div class="sb-col">
        <sb-alert type="warning">This is an warning alert.</sb-alert>
      </div>
      <div class="sb-col">
        <sb-alert type="error">This is an error alert.</sb-alert>
      </div>
    </div>
  </div>
</template>
<style scoped>
.sb-row {
  display: flex;
  flex-flow: row wrap;
  justify-content: flex-start;
  align-items: flex-start;
}

.sb-col {
  box-sizing: border-box;
  flex: 0 0 50%;
  width: 50%;
  padding: 10px 20px;
}
</style>

可关闭 #

通过设置 closable，可开启关闭按钮。

This is an info alert.
This is an success alert.
This is an warning alert.
This is an error alert.
<template>
  <div class="box">
    <div class="sb-row">
      <div class="sb-col">
        <sb-alert v-model:visible="visible1" @click="handleClose(visible1)" closable>This is an info alert.</sb-alert>
      </div>
      <div class="sb-col">
        <sb-alert v-model:visible="visible2" @click="handleClose(visible2)" type="success" closable>This is an success
          alert.</sb-alert>
      </div>
      <div class="sb-col">
        <sb-alert v-model:visible="visible3" @click="handleClose(visible3)" type="warning" closable>This is an warning

          alert.</sb-alert>

    </div>
  </div>
</template>

操作项 #

通过 #action 插槽自定义操作按钮。

This is an info alert.
Example
This is an info alert. 
<template>
  <div class="box">
    <div class="sb-alert-space-vertical" style="width: 100%;">
      <sb-alert v-model:visible="visible1" @click="handleClose(visible1)"  closable style="margin-bottom: 24px;">
        This is an info alert.
        <template #action>
          <sb-button size="small" type="primary">Detail</sb-button>
        </template>
      </sb-alert>
      <sb-alert  v-model:visible="visible2" @click="handleClose(visible2)"  title="Example" closable>
        This is an info alert.
        <template #action>
          <sb-button size="small" type="primary">Detail</sb-button>
        </template>
      </sb-alert>
    </div>
  </div>
</template>

<script lang="ts" setup>
import { ref } from 'vue'
const visible1 = ref(true)
const visible2 = ref(true)
const handleClose = (visible: any) => {
  visible.value = false;
}
</script>

<style scoped>
.sb-alert-space-vertical {
  display: inline-flex;
  flex-direction: column;
}
</style>

顶部公告 #

通过设置 banner，可将警告提示作为顶部公告使用（去除边框和圆角）。

This is an info alert.
<template>
    <div class="box">
        <sb-alert banner>This is an info alert.</sb-alert>
    </div>
</template>

API #

<alert> Props #

参数名描述类型默认值
type警告提示的类型。infosuccess
show-icon是否展示图标booleantrue
closable是否展示关闭按钮booleanfalse
title警告提示的标题string-
banner是否作为顶部公告使用（去除边框和圆角）booleanfalse

<modal> Events #

事件名描述参数
close点击关闭按钮时触发ev: MouseEvent
after-close关闭动画结束后触发-

<alert> Slots #

插槽名描述参数
icon图标-
title标题-
action操作项-
close-element关闭元素-
